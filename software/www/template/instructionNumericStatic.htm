This password system uses a numeric challenge grid (top) containing the numerals 1 through 9.
The password is shown at the bottom and it will always be four digits in length, composed of numerals present in the challenge.
Your task is to enter the password shown by clicking the elements of the challenge grid.

For example, the password is {9,2,5,7}.

You would first select the 9 by clicking the square containing that numeral,
since 9 is the first digit of the password.

Followed by the 2, 5, and 7.
When you are ready to submit the password, click the SUBMIT button.

If you enter the correct password, Correct will be displayed for 0.5 seconds, after which a new challenge and password will then be presented.

Otherwise, Incorrect will be displayed for 0.5 seconds. The challenge and password will remain the same.

If you exceed the password length limit of 4, a message will be displayed for 0.5 seconds.
If you make a mistake, you can clear your response by clicking the CLEAR button.

This concludes the instructions. Please click DONE to continue.
