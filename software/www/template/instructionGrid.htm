This password system uses a challenge grid (top) containing a random ordering of shapes, shape-exterior colours, and shape-interior colours.

The password is shown at the bottom. It will always be four digits in length. A completely grey shape represents a shape attribute (E.g. square, circle), where a partially coloured star will represent either an exterior, or interior colour attribute (depending on which corresponding part of the star is coloured).
Your task is to enter the password shown by clicking the elements of the challenge grid that contain the attribute for that password digit.

In this example, the password is {blue-interior,down-triangle,purple-interior,green-exterior }.

You would first select the grid element containing the right-triangle with blue-interior and green-exterior ,
since the first password digit is: blue-interior.

Followed by the down-triangle with purple-interior and purple-exterior,
since the second password digit is: down-triangle.

Followed by the down-triangle with purple-interior and purple-exterior,
since the third password digit is: purple-interior.

Followed by the right-triangle with blue-interior and green-exterior ,
since the fourth password digit is: green-exterior.
When you are ready to submit the password, click the SUBMIT button.

If you enter the correct password, Correct will be displayed for 0.5 seconds, after which a new challenge and password will then be presented.

Otherwise, Incorrect will be displayed for 0.5 seconds. The challenge and password will remain the same.

If you exceed the password length limit of 4, a message will be displayed for 0.5 seconds.
If you make a mistake, you can clear your response by clicking the CLEAR button.

This concludes the instructions. Please click DONE to continue.
